# Boot 全自动工具箱
by aming1029

Windows 端一键 ROOT 工具箱，支持：
- 多版本面具（官方 / Alpha / Kitsune / FolkPatch / SK / APatch）
- 一键提取 Boot / init_boot
- 一键修补
- 一键刷入
- 一键 ROOT（全自动）
- AK3 云端下载
- 隐藏环境启动
- 粉白美观 GUI

## 使用方法
1. 从 Actions 下载编译好的 magiskboot
2. 放入目录
3. 运行 build.bat 打包单 EXE
4. 打开工具箱 → 一键 ROOT
